## 핵심 정보 요약

### API 기본 구조
- **Base URL**: `https://api-gateway.coupang.com`
- **인증**: OpenAPI Key (HMAC 서명 방식)
- **지원 지역**: 한국 (일부 API는 대만 지원)

### API 호출 정책
- API 호출 빈도 제한이 있음 (Rate Limiting)
- 과도한 호출 시 자동 차단 가능
- 2026년 3월 15일부터 고객 문의 API 응답에서 buyerEmail 필드 삭제

### API 카테고리 전체 목록
| API 카테고리 | 주요 용도 |
|-------------|----------|
| 카테고리 API | 상품 카테고리 조회 |
| 물류센터 API | 출고지/반품지 관리 |
| **상품 API** | **상품 CRUD, 조회, 검색** |
| 배송/환불 API | 주문 배송 및 환불 처리 |
| 반품 API | 반품 접수 및 처리 |
| 교환 API | 교환 접수 및 처리 |
| 쿠폰/캐시백 API | 프로모션 관리 |
| **CS API** | **고객 문의 조회/답변** |
| 정산 API | 정산 내역 조회 |
| **로켓그로스 API** | **로켓그로스 상품/주문/재고** |
